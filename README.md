# idea-settings
### Appearance & Behavior

#### Appearance

- theme: IntelliJ
- windows options: 勾选 show memory indicator
- windows options: 勾选 small label in editor tabs
- System Settings: 取消勾选 Reopen last project on startup

### Editor

#### Font

- Font：Hack: https://sourcefoundry.org/hack/
- Font: Hack, 12, 1.2
- fallback font: 微软雅黑，备用字体，显示中文

#### Color Scheme

- save as ...tom
- General - Text - Default Text：Background #F2F2F2

#### File encodings

- Global encoding: UTF-8
- Project encoding: UTF-8
- Default encoding for properties files: UTF-8
- 勾选 Transparent native-to-ascii conversion

### Build, Execution, Deployment

#### Compiler

- 勾选 Build project automatically（自动编译）
- 按 ctrl+alt+shift+/，选择Registry，勾选 compiler.automake.allow.when.app.running（自动编译）
